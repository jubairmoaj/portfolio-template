# Quick Math Challenge

## Overview
Quick Math Challenge is a compact, addictive app for practicing arithmetic and problem solving with timed rounds, leaderboards, and achievement progression.

## Key Features
- Multiple difficulty levels and timed rounds
- Daily challenge and streak tracking
- Offline play with local score caching
- Progress/XP system and basic leaderboards

## Tech Stack
- Flutter (UI & cross-platform clients)
- Kotlin Multiplatform for shared logic (optional modules)
- Firebase for cloud sync and analytics
- Node.js backend for leaderboards and moderation

## Impact
- Used by learners to build speed and accuracy with short practice sessions.
- Part of AppPulse’s combined Play Store presence (1M+ installs across apps).

## Future Plans
- Add multiplayer head-to-head timed matches and seasonal challenges.
