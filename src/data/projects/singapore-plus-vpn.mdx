# Singapore Plus VPN

## Overview
Singapore Plus VPN is a focused mobile VPN product that prioritizes speed and privacy while offering an effortless connection experience targeted at users needing Singapore/Asia routing.

## Key Features
- One-tap connect / disconnect
- Multiple server regions with Singapore servers prioritized
- Split tunneling (select apps to use VPN)
- Simple, minimal UI built with Jetpack Compose
- Connection stats and session logs
- Built-in diagnostics for connectivity

## Tech Stack
- Kotlin, Jetpack Compose
- Node.js backend (server management and analytics)
- Firebase (analytics, crash reporting)
- OpenVPN / WireGuard backend integrations
- Figma for UI, Git for source control
