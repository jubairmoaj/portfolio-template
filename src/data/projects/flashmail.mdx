# FlashMail

## Overview
FlashMail provides users with instant disposable email addresses to sign up for web services without exposing their primary inbox.

## Key Features
- Create and copy disposable addresses in one tap
- Multiple temporary inboxes
- Push notifications for incoming messages
- Automatic expiry and deletion policies
- Lightweight UI and small binary footprint

## Tech Stack
- Android (Java + Kotlin)
- Firebase for push notifications and analytics
- Node.js + MySQL for mailbox management
- Admin panel for monitoring and abuse controls

## Impact
- Widely used for spam protection and privacy-first signups.
- Part of AppPulse’s combined Play Store audience (1M+ across apps).

## Future Plans
- Add encrypted mailbox option and paid tier for extended retention.
